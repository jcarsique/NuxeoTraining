Installation et démarrage

Téléchargement

Disponible depuis nuxeo.com : http://www.nuxeo.com/fr/telechargez-nuxeo

Packages disponibles:

  • Package multi-plateforme (.zip) : Serveur Nuxeo complet zippé
  • Installeur Windows (.exe) : installe les dépendances si nécessaire (Java, OpenOffice, etc.)
  • Installeur Debian/Linux : installe les dépendances et un script init
  • Images de machine virtuelles

Toutes les procédures d'installation : doc.nuxeo.com

Installation

Prérequis : Java 7

Où : Racine d'un disque (C:\ par exemple)

  • Pas de problème de profondeur de chemin
  • Pas de problème de chemin avec des espaces / caractères spéciaux

Quoi : Serveur Nuxeo

Démarrage

Double-clic sur Start Nuxeo.bat

-> Panel Control pour arrêter le serveur

URL: http://localhost:8080/nuxeo

Droits d'accès

Droits d'accès

Définition

Ensemble de permissions qui détermine les actions disponibles. Les droits d'accès peuvent être accordés ou refusés.

Pour qui

Déterminés pour :

  • utilisateur individuel
  • groupe d'utilisateur

Où ça

Appliqués sur les documents “folderish” = pouvant contenir d’autres documents

  • Domaine
  • Espace de travail, de publication
  • Dossier
  • ...

Workflow en série

Ordre d'intervention des participants

Simple revue ou validation

  • Pas de changement d'état
  • Projet -> Valide

Workflow en parallèle

Objectif : valider ou non un document après une étape de consultation auprès de plusieurs participants

Parallèle car pas d'ordre d'intervention des participants

Déroulement :

  1. Initiateur demande leur avis à des participants
  2. Les participants indiquent : OK, pas OK, sans opinion
  3. L'initiateur a le résumé de la consultation et décide de valider ou non le document.
