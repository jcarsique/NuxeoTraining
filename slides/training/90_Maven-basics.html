# Nuxeo training - Maven Basics

Maven?

### Maven? #### Concepts - Software tool for: - Java project management - Build automation - Uses Project Object Model (POM) to describe the project being built - Dynamically downloads Java libraries - Use Maven 2.2.1 with Nuxeo (problems with Nuxeo and Maven > 2.2.1)
### Why do we use Maven? #### Building Nuxeo software is complex - Lot of components / modules (500+) - Complex dependencies #### Packaging Nuxeo software is complex - Lots of components - Packaging depends on target server layout #### Release process is complex
### What does Maven provide? - Declarative dependency management - Full build automation - Snapshot management via Maven repository - Pluggability
### Maven Basics - Build file is pom.xml - groupId / artifactId: identify artifact - parent pom: common dependency - dependencies ##### Useful commands: 
mvn install
mvn clean install
mvn test
mvn -Dmaven.test.skip=true goal
mvn eclipse:eclipse
### Maven Phases - Main phases for Maven: - Validate - Compile - Test - Package - Install (in the local repository `~.m2/repository`) - Deploy
