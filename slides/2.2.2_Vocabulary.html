### Vocabularies List of items used in the application - Id/code
ID: ID of the reference. This is what is stored on the document
- Value/Label
Label: The value that is displayed in the UI
- Obsolete
Obsolete: if true, the value is not shown in the drop down list, but still computed for documents that used the entry
- Order
Integer used in some widget to order the list
By default stored in the database - 1 table 1 vocabulary - Entries can be modified by administrators
Admin Center, Vocabularies
- Widget can pick values from it: Coverage, Subjects
