Java operations in Nuxeo Studio

  • Lots of available operations, but it may not be enough
  • Studio for chains configuration, Java for advanced customization
    • Access to services
    • Custom behaviour that can't be generic and integrated in Studio by Nuxeo 
    • Missing operations in Studio, can't wait for the next release

Create a java operation

  • Create a standard java class

    • The class annotated with @Operation  
    • Runnable method with @OperationMethod

      • Multiple operation methods (with different signatures) 
      • Inputs as method parameters 
      • Output as returned object
    • Operation parameters with @Param on attributes

Operation sample

      
@Operation(id = SampleOperation.ID, category = Constants.CAT_SERVICES, label = "Sample Operation", description =
"An example.")

public class SampleOperation {
  public static final Log log = LogFactory.getLog(SampleOperation.class);
  public static final String ID = "Sample.Operation";

  @Context
  protected CoreSession session;

  @Context
  protected OperationContext ctx;
  
  @Param(name = "type")
  protected String type;

  @Param(name = "name", required = false)
  protected String name;
   
  @OperationMethod
  public DocumentModel run(DocumentModel doc) throws Exception {
    log.info("runing sample operation with params: type=" + type + "; name=" + name);
    return doc;
  }
}

Register the operation

A extension point to register the operation

      
<extension
  target="org.nuxeo.ecm.core.operation.OperationServiceComponent"
  point="operations">
  <operation class="org.nuxeo.example.TestOperation" />
</extension>

Chain the operation in Studio

Nuxeo Studio needs to know the definition of your custom operation.

  • http://localhost:8080/nuxeo/site/automation/doc
  • Copy the JSON definition of the custom operation
  • Nuxeo Studio > Advanced settings > Registries > Operations
  • Paste the JSON definition surrounded with  
              
{
 operations: [ ... ]
}

The operation is available in the operations list in Studio
