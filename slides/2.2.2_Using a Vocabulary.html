# Nuxeo Training - Vocabularies ######Sun Seng David TAN
### Vocabularies List of items used in the application - Id/code - Value - Obsolete - Order By default stored in the database - 1 table 1 vocabulary - Can be modified by administrators - Widget can pick values from it: Coverage, Subjects
### Several kind of vocabularies - Simple Vocabulary - Child Vocabulary  - One more column: parentId - Both in the same Vocabulary: hierarchical 
### Vocabulary management - In Nuxeo Admin Center - can manage existing vocabularies - can not create new ones
### Vocabulary creation policy - Creates from the CSV file  - On missing columns  - Table structure changes  - First time - Always - Never 
### Vocabularies in layouts Widgets in studio For text field - Simple vocabulary - 2 levels vocabularies - For multivalued text field - Simple vocabulary - 2 levels vocabularies - No 3+ levels vocabularies ! - Template widget
