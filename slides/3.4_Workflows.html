Workflow management in Nuxeo UI

    • currently started workflows
    • Waiting task for this doc + layout + actions
  • Dashboard
    • Waiting tasks for the user + link to the doc
  • Admin Center
    • Workflow templates availables
  • Coming soon: Workflow tab
    • Pending tasks

Concepts

  • Node: square, contains attributes
    • Task: A node can create a Task (A NodeTask) -> will create a task (from Nuxeo), but can type the task to add metadata. A node task can have assignee and will wait for the action of the user
    • Start node
    • End node
  • Transition: point -> arrow. Can bind to automation chains. Condition to take the transition (or not)
  • Workflow Variables -> will be accessible during the worflow (persisted). Actually fields of the workflow
  • Node variable -> only visible by the node or can be visible (persisted). Actually fields of the task
  • A node can accept a form layout -> will edit node variables OR workflow variables can set to a form, actions
  • Can bind automation chains everywhere -> can set workflow variables or node variables
  • Queues (not exposed in Studio yet) Content views of waiting tasks -> needs mapping to transfer properties existing doc. Bulk process of all the task of a node of a workflow
